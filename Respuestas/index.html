Analizador de Palabras

Ciencias de la Computación: Estructuras de Datos

Proyecto de estructuras de datos. Analizador de palabras, que realiza el conteo de palabras por archivo y en número de apariciones. Además, la gráfica que se presenta abajo indica cuando dos archivos tienen palabras en común con una longitud de caracteres mayor a 7.

Estructuras de Datos utilizadas:


  1. Diccionario: Se utilizó para hacer el conteo de apariciones por palabra.
  2. Listas ligadas: Se utilizaron para ordenar las llaves de los diccionarios.
  3. Árboles Rojinegros: Se utilizaron para desplegar las apariciones de palabras.
  4. Árboles Autobalanceables (AVL): Se utilizaron para desplegar las apariciones de palabras.
  5. Conjuntos: Se utilizaron para hallar la palabras en común entre archivos.
  6. Gráficas: Se utilizaron para representar a los archivos que comparten palabras.

Archivos que se analizaron:

  1. Ejemplos/amante.py tiene 84 palabras.
  2. Ejemplos/amor.txt tiene 2 palabras.
  3. Ejemplos/entendimiento.c tiene 83 palabras.
  4. Ejemplos/hombres.txt tiene 313 palabras.
  5. Ejemplos/ilision.html tiene 169 palabras.
  6. Ejemplos/retrato.csv tiene 86 palabras.
  7. Ejemplos/rosa.ghci tiene 93 palabras.
